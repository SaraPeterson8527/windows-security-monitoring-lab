# Windows Security Monitoring Lab

## Overview

This project demonstrates a hands-on Windows security monitoring environment designed to practice system logging, event analysis, and incident investigation.

The goal of this lab was to gain practical experience reviewing Windows security events, identifying suspicious activity, and documenting findings in a format similar to a Security Operations Center (SOC) workflow.

---

# Objectives

- Configure a Windows monitoring environment
- Review Windows Event Logs
- Identify important security events
- Analyze user activity and system changes
- Document investigation findings

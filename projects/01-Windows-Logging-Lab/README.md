# Project 01 - Windows Logging Lab

## Objective
Prepare a Windows 11 virtual machine for SOC monitoring and learn how Windows records system activity.

## Environment
- Host: Macbook Air M4
- Virtualization: UTM
- Operating System: Windows 11 ARM

## Goals
- Learn Windows Event Viewer
- Understand Windows security logs
- Install Sysinternals tools
- Deploy Sysmon
- Analyze system activity
